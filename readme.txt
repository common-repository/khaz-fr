== Description ==

Khaz-fr est un outil BtoB révolutionnaire qui identifie les entreprises visitant votre site web, et ce plugin WordPress facilite son intégration sans effort à votre site. Avec une installation en quelques clics, libérez le potentiel de votre site web pour générer 10 fois plus d'opportunités de business sans modifications supplémentaires.

== Installation automatique : ==
Cliquez sur installer ici
Cliquez ensuite sur activé

==Installation manuelle :==
Téléchargez le fichier « http://khaz-fr.zip ».
Extrayez le contenu du fichier zip.
Téléchargez le dossier « khaz-fr » dans le répertoire « /wp-content/plugins » de votre site WordPress.
Accédez à l'écran des plugins dans votre tableau de bord WordPress.
Localisez le plugin Khaz-fr et cliquez sur Activer.

==Avis de confidentialité==
Ce plugin peut automatiquement collecter et stocker les informations suivantes :
- Adresse IP
- Localisation
- Détails matériels et logiciels
- Données des visiteurs de votre site web (adresse IP uniquement, via l'installation d'un script de votre part)
Ces données peuvent être collectées par les méthodes suivantes : Connexion aux données de votre site web via un script à intégrer par vous sur votre site. En ce qui concerne les outils tiers, l'application utilise les API des fournisseurs de ces services tiers pour récupérer, stocker, traiter et créer des données. C'est notamment le cas des API Pappers (données administratives) et des API des bases de données open source gouvernementales. Veuillez noter que nous ne collectons que les données qui nous aident à atteindre l'objectif énoncé dans cette politique de confidentialité. Nous ne collecterons pas de données supplémentaires sans vous en informer au préalable. Veuillez noter que nous ne collectons et ne vous fournissons que les données de vos visiteurs professionnels. Les connexions potentiellement privées (depuis des mobiles, des box privées, etc.) ne sont ni rapprochées ni enrichies, en conformité avec les règles sur les données personnelles. Veuillez noter que http://Khaz.fr n'utilise aucun cookie pour les visiteurs de votre site. Aucun suivi n'est donc effectué de cette manière.
Les données personnelles collectées par le script ne seront utilisées que pour les finalités spécifiées dans cette politique ou indiquées sur les pages pertinentes de notre site. Nous n'utiliserons pas vos données au-delà de ce que nous divulguons.
Les données que nous collectons automatiquement sont utilisées aux fins suivantes :
- Collecter des données sur l'utilisation des dispositifs web des utilisateurs de la plateforme
- Fournir aux utilisateurs de la solution des informations et des données pertinentes dans leur cas particulier
- Identifier et sécuriser l'accès à leur compte d'application
- Permettre le paiement facile de l'abonnement à l'application pour les utilisateurs payants qui le souhaitent
